Finger Controlled Wheel-Chair using Arduino & Flex sensor


This was an intership project

This project describes a wheelchair for physically disabled people. The goal is to design and develop a system that allows the user to robustly interact with the wheelchair at different levels of the control and sensing.

A dependent-user recognition using finger movements and infrared sensor integrated with wheelchair. Wheelchair which can be driven using DC Driver Motor with separate power supply and finger Movements with the possibility of avoiding obstacles by using obstacles sensors.

Note: This was prototype model that we built to showcase our idea to client

Home appliances System

Home appliances system made form vb.net with the database of MS ACCESS

This project was built in the framwork Microsoft Visual Studio 2010. It was an small project containing information about daily transaction a small scale business goes through.

it holds the records like purchase, sales, stock at hand, stock which is going to be sold on a particular day, cutomers record/information, employees records/details etc.

It has database of MS Access. We had connected that database to the front-end using in-built services from MS Visual studio 2010. This database basically had tables associated with each forms and tables has fields associated with fields in the form. In this way the entered record/information gets stores at their intended location.
This app was able to generate reports whenever required. This service was available only for the internal employees.
